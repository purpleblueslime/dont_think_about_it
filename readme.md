# build yourself 
**fork** and **pull**.
```sh
-> git add origin https://github.com/purpleblueslime/dont_think_about_it.git 
-> git pull origin
```

## install packages
```sh
-> flutter pub get
```

## icon
```sh
-> dart run flutter_launcher_icons
```

## run
```sh
-> flutter run app.dart
```

## ios
```sh
-> flutter build ios -t app.dart
```

## android
```sh
-> flutter build apk -t app.dart
```
